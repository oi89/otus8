Проект с использованием Allure отчетов.

Реализовано снятие скриншотов:
- скриншоты снимаются на каждый шаг теста
- скриншоты снимаются при падении теста:
    для этого реализован интерфейс TestWatcher, который через аннотацию подключается к классу BaseTest

Использование:
- запустить тесты "mvn clean test"
- сгенерировать Allure-отчет (в консоли "mvn allure:serve" или через allure плагин)
